# Page Factory

The page factory turns Workpedia data into public pages.

## Local Service Page Template

Use for URLs like:

- `/locations/kitchen-remodel-frisco-tx.html`
- `/locations/bathroom-remodel-mckinney-tx.html`
- `/locations/painting-contractor-denton-tx.html`

Required sections:

1. Hero: service + city + one local reason.
2. Local context: housing stock, neighborhood pattern, or climate issue.
3. Scope tiers: small, standard, premium.
4. What changes price.
5. Prep checklist.
6. Common local failure modes.
7. FAQ.
8. CTA.
9. Workpedia module: cards, facts, request buttons, best first message.

## Guide Page Template

Use for URLs like:

- `/guides/kitchen-remodel-cost-dfw.html`
- `/guides/types-of-interior-paint-finishes.html`

Required sections:

1. Direct answer.
2. Table or checklist.
3. Decision model.
4. Mistakes to avoid.
5. Local DFW note.
6. Related guides.
7. CTA.

## Data Requirements Before Publishing

- Title.
- Meta description.
- Canonical.
- Breadcrumb schema.
- FAQ schema if there are at least three FAQs.
- Internal links to at least two services, two locations, and two guides where relevant.
- Sitemap entry.
