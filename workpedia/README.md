# Workpedia

Workpedia is the internal encyclopedia for ELF Home Services. It is not a public page yet. It is the knowledge library used to build service pages, local pages, guides, FAQs, calculators, snippets, and future interactive tools.

## Purpose

Workpedia should make the site feel deep, useful, local, and practical. Every entry should help answer one of these questions:

- What is the homeowner trying to decide?
- What can go wrong if this work is done badly?
- What should be measured, photographed, tested, or chosen before quoting?
- What is specific to North Texas, DFW, Denton, Frisco, Plano, McKinney, Allen, Lewisville, Flower Mound, Southlake, or Dallas?
- What can be turned into a checklist, calculator, comparison table, FAQ, or local guide?

## Volumes

- Volume 01: Foundation, URL inventory, taxonomy, service facts, local signals, page factory.
- Volume 02: Cost guide data and calculators.
- Volume 03: Local guide expansion for the 90 upcoming URLs.
- Volume 04: DIY and homeowner decision libraries.
- Volume 05: FAQ, schema, snippets, and internal-link graph.
- Volume 06: Visual asset prompts, comparison diagrams, and page modules.

## Source Rule

Use Workpedia as structured planning knowledge. Public claims that depend on current law, permits, prices, code, market conditions, or product availability should be verified before publishing as final copy.
